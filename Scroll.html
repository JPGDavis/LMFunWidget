Monitor Everything, Everywhere

Dashboards don’t need to be boring! This is a dashboard showing how custom HTML can blend seamlessly with the rest of your reporting.

In this example, the HTML widget in the center can be used to shoot down approaching TIE Fighters.

We’re using HTML5 canvas line drawing to give everything the retro feeling of the classic Star Wars arcade games.

Metrics for ships shot down are collected by a LogicModule polling a web server and stored within LogicMonitor. These are displayed via our graphing widgets to the left of the page.

Alerts coming into the widget on the right are sourced from a live host via a custom scripted EventSource. In this example we’re pulling quotes from the famous battle in Star Wars Episode IV, but in your setup these can be anything...
